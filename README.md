# IntentShare
这是一个关于本地分享的项目；提供使用Intent分享微信、QQ、QQ空间、新浪微博的一些方法。


[关于这个项目的一些简单介绍](https://www.jianshu.com/p/9522e24713e1)

### 基本说明  
关于微信分享

``` 
1.微信在6.6.7版本以后，已经不支持往朋友圈分享标题了  
2.微信在6.7.3版本以后，已经不支持往朋友圈分享多图了 
```
